# VotV Points Editor

A utility tool for editing and customizing save data and points in *Voices of the Void* (VotV).

## Features

- 🛠️ Simple GUI to modify game points and save values.
- 💾 Automatic backup before writing file changes.
- ⚡ Fast execution with zero dependencies.

## Usage

1. Launch `VotV-Points-Editor`.
2. Select your VotV save file location.
3. Adjust point values and click **Save**.
